#Number Guessing Game (Python):
A simple and interactive Number Guessing Game built using Python.
The program randomly selects a number, and the user has to guess it within minimum attempts.

##🚀 Features
🎲 Random number generation
🔢 User input handling
📉 Hints (Too High / Too Low)
🔁 Multiple attempts tracking
🏆 Displays number of tries taken to guess correctly

##🛠️ Technologies Used
Python (Core Concepts)
Random Module

##▶️ How to Run
Clone the repository:
git clone https://github.com/your-username/repo-name.git
Navigate to the project folder:
cd Number-Guessing-Game
Run the program:
python game1ui.py

##🎮 How to Play
The system will generate a number between 1 and 100
Enter your guess in the terminal
You will get hints:
📈 "Go Higher"
📉 "Go Lower"
Keep guessing until you find the correct number

##📌 Example Output
Guess number between 1 to 100: 50
Go Higher
Guess number between 1 to 100: 75
Go Lower

Congratulations! You found the number in 3 tries.
